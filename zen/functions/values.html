Function values

  • values<Tinput, Toptions>(input, options?): Values<Tinput, Toptions>[]

  • Returns an array of either OR both of: a) Nested / Nested Values (eg Array elements, Map values, Object.values() etc) b) the props Values (i.e _.values() equivalent) of the input value, according to KeyOptions. Rigorously typed according to the input value and KeyOptions.

    The features, logic & options (of KeysOptions) are identical to z.keys, but it returns the nested/props values instead of the nested keys/props. See Keys for more details.
